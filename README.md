# About ProjectWP

ProjectWP is a web applicaiton where you can buy and a variety of digital games running on PC through our web application built with Laravel and MySQL

## Requirements

-   Laravel v8.0 or above
-   Composer v2.0 or above
-   PHP v7.4.29 or above
-   MySQL & Apache

## Installation

```sh
git clone https://github.com/madaghafuri/Project_WP.git
cd Project_WP
composer install
php artisan serve
```
